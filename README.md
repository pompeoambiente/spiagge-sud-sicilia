# Spiagge Sud Sicilia

Sito statico pronto per GitHub Pages.

## Pubblicazione rapida

1. Crea un nuovo repository GitHub, ad esempio `spiagge-sud-sicilia`.
2. Carica nella root del repository questi file:
   - `index.html`
   - `.nojekyll`
   - `README.md`
3. Apri `Settings -> Pages`.
4. In `Build and deployment`, scegli:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Salva e apri il link generato da GitHub Pages.

## Nota

La geolocalizzazione mobile richiede HTTPS. GitHub Pages fornisce HTTPS sul dominio `github.io`.
